# PageSpeed submission note

Because PageSpeed Insights requires a hosted URL, host the landing page temporarily using one of these options:

- GitHub Pages
- Netlify Drop
- Vercel static deployment

After hosting:
1. Open PageSpeed Insights.
2. Test the live landing page URL on **Mobile**.
3. Capture the screenshot showing the score.
4. Add the screenshot file to the repository before final submission.

## Why this page should score well

- Single HTML file
- No frameworks
- No external images
- Minimal JavaScript
- Inline CSS with no render-blocking layout complexity
- System-font fallbacks after a single fast font request
